# About This Playground

Ohajiki is a game app based on the traditional Japanese game of Ohajiki. By blending digital expression with the classic Ohajiki, this app aims to be accessible to all, from children to the elderly, thereby changing the current situation where ohajiki is gradually being forgotten. Furthermore, it seeks to harness the inherent benefits of Ohajiki: promoting child development, preventing dementia in the elderly, and fostering intergenerational exchange.
